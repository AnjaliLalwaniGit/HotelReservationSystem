# HotelReservationSystem
The hotel reservation application allows customers to find and book a hotel room based on room availability.
This project has been implemented in Java and was developed as a part of the Java Developer coursework in Udacity.
## User Scenarios

The application provides four user scenarios:

#### Creating a customer account
#### Searching for rooms
#### Booking a room
#### Viewing reservations

## Admin Scenarios

The application provides four administrative scenarios:

#### Displaying all customers accounts.
#### Viewing all of the rooms in the hotel.
#### Viewing all of the hotel reservations.
#### Adding a room to the hotel application.

### Usage
The application needs to be run from command line and the user needs to make one of the selections from the menu item.
